Analysis Workflow

From synthetic cascade generation to interpretable rumor prediction.

01

Generate a temporal tree

A directed propagation tree is sampled as rumor-like or organic-like with edge-level delay and timestamp attributes.

02

Compute graph features

Structural and temporal metrics are extracted, including diameter, radius, center, density, and average temporal weight.

03

Run classifier inference

The RandomForest model maps the feature vector to rumor or organic class and returns confidence.

04

Interpret propagation behavior

Use edge tooltips, prediction confidence, and summary cards to explain the result with graph theory concepts.

How to read the graph

  • Node 0 is the source vertex where diffusion starts.
  • Thicker edges represent stronger temporal weight from faster repost delay.
  • Diameter and radius summarize global path spread in the undirected view.
  • Center identifies structurally central vertices with minimum eccentricity.
  • Prediction confidence indicates how strongly features match known patterns.

Feature groups used by the model

Size and shape

Nodes, edges, average degree, max degree, branching factor, and leaf ratio.

Path structure

Density, diameter, radius, center, and average shortest path length.

Centrality profile

Degree centrality, betweenness, closeness, and overall centralization.

Temporal behavior

Per-edge temporal weighting, average temporal weight, and diffusion speed.
