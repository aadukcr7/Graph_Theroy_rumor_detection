About the Project

Why propagation structure matters

This project studies how rumors and organic information spread through temporal propagation trees. It uses interpretable graph features rather than a graph neural network, which keeps the method simple, explainable, and suitable for a final-year graph theory project.

Method

Graph generation

Rumor graphs are generated as star-like, hub-heavy, and rapidly branching trees. Organic graphs are generated as paths and sparse chains.

Temporal modeling

Each edge carries a delay and a timestamp. Fast reposts receive larger temporal weights, so the model sees both structure and speed.

Classification

A RandomForestClassifier learns from the extracted graph metrics and predicts whether a cascade looks like rumor or organic information.
